## 项目介绍

低代码问卷项目，服务端部分

- 答题端（C端）
  [survey-client](https://github.com/yangfei4913438/survey-client)

- 设计端 (B端)
  [survey](https://github.com/yangfei4913438/survey)

#### 运行方法

##### 1、启动服务端

```bash
# 安装依赖
yarn

# 运行开发服务
yarn start:dev
```

##### 2、启动设计端（B端）

```bash
yarn
yarn dev
```

##### 3、启动答题端（C端）

```bash
yarn
yarn dev
```
